怎样使用Octopress来写博客

Jul 5th, 2013

需要使用的命令：

  • chcp 65001
  • rake new_post[“Zombie Ninjas Attack: A survivor’s retrospective”]
  • rake generate
  • rake preview
  • rake deploy

安装通过Railsinstaller（注意：路径不要带空格）。gem uninstall rake删除默认版本，并安装gem install rake -v’0.9.2.2’。
